## go math, 数学计算

## 除法

```go
package main

import "fmt"

func main() {
  	fmt.Println(1/2) 
	fmt.Println(2/2)
	fmt.Println(3/2)
	fmt.Println(4/2)
}

```

## 求余, 取模

```go
package main

import "fmt"

func main() {
  	fmt.Println(1%2)
	fmt.Println(2%2)
	fmt.Println(3%2)
	fmt.Println(4%2)
}
```
