## CSS注释语法

    /* 注释内容 */

### 示例

    /* ----------文字样式开始---------- */
    /* 梦之都白色12象素文字 */
    .dreamduwhite12px
    {
     color:white;
     font-size:12px;
    }
    
    /* 梦之都黑色16象素文字 */
    .dreamdublack16px
    {
     color:black;
     font-size:16px;
    }
    /* ----------文字样式结束---------- */
