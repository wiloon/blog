W25 把 enx 的 Cognito 分支合进 main、Java 后端方向定了下来，脚手架和落地留到了这周。下面记 W26 的事。

## enx 加 Java 后端

最近投了 Java/Spring 方向的职位，这段时间在回忆和熟悉 Java、Spring，也在整理 blog 里攒下的 Java/Spring 资料。但只看文章、不动手，理解很有限。过去十几年用 Java 训练过的那套"生物神经网络"痕迹应该还在，细节被修剪掉了，主要的连接还在，得花点时间把细节重新接上——而重新接上的最好办法是有真正的代码跑起来。

enx 是我自己用的技术 playground，正好拿它练手。这周给 enx 加了一个 Java/Spring 后端，把一个简单的 API 迁了过去。技术栈大致是 Spring Boot，安全这块复用 enx 已有的 Cognito，依赖走 Homelab 里的 Nexus Maven 私服，构建和部署接到现成的 Tekton + k8s 流水线上：推代码后 Tekton 自动打镜像、更新到集群。这样一个最小闭环先跑起来，后面再慢慢补。

为什么要这么折腾一个个人项目，其实和求职是一条线：与其等机会确定了再准备，不如先把状态备好。四年前进上一家公司前后，我也是在面试前、拿到 offer 后、入职前一直在补 Python，正因为有前期铺垫，入职一个月就能在团队协助下独立交付一个用户需求。这次 Java 也一样，机会本身是不确定的——大厂开一个岗会收到海量简历，能不能轮到面试、会不会有更合适的人，都不是我能决定的。我能做的只是让自己在机会出现时是有准备的状态。

## 梳理 Java/Spring 文档

接着 W25 的节奏，继续整理 blog 里的 Java 笔记，这周重心偏到 Spring 和 JVM 周边。大脑像是索引和内存，blog 像是外部存储；脑子里记得的大概是 blog 内容的一个子集加上索引，有交叉，但不是所有东西都写进了 blog。整理的过程就是顺着索引把外存里的细节重新调出来。

照例不在周记里展开技术点，只记几篇本周动过的，方便以后自己回溯：classloader 与 classpath 一组、Spring IoC 与 Spring Boot、graalvm native image、HotSpot，以及顺手补的 SPI / ServiceLoader 和 yagni。

## 公司：领照、刻章、开户

W24 注册了一人公司，这周把后续手续办了：领营业执照、刻章、银行开户。本来 6 月 24 日计划去领照，车启动打开导航才发现当天大连有达沃斯论坛限行，只好折回，顺便在家准备面试和注册材料。第二天才正式跑完一圈。

过程中联系上了海创 OPC（一人公司）社区，工作人员介绍了入驻的优惠政策：token 补贴、项目奖励，做得好也许还有投资；入驻条件是用一页纸说清楚要做什么，再准备五分钟左右的 PPT 讲公司和项目的价值——某种意义上像是另一种"面试"。社区有合作银行和免费刻章，税务报到也不强制 30 天内完成，可以延到 90 天、在有实际业务之前再办。我一度担心入驻流程太长会拖慢开户时间，社区的人还帮忙对接了银行的优惠。

有时候觉得太折腾了，但第二天早上醒来又想，已经走到这一步，就再往前探索一下。换个角度，这也是在对抗裁员之后的不确定性。这段时间不像在职时——有人帮忙梳理需求，出了问题有领导去跟客户解释；现在所有事只能靠自己。但也因此接触到很多不同角色的人：政府的、OPC 社区的、银行的，大家都挺友善地给建议和帮助。政府部门也不再是印象里冷冰冰的样子，工作人员会在规则允许的空间里尽量提供支持，税务上也有弹性。裁员把人推出舒适区，到丛林里重新找机会，倒也不全是坏事。

## 用 AI 办线下的事

这段时间遇到不熟悉的领域，基本是"遇事不决问 AI"。海创 OPC 社区这条线索就是 AI 给的，虽然它给的电话打过去是家酒店，但地址给对了。也不能全信AI, AI 说大连高新区行政大厅免费刻章，跑过去问，对方说这政策很早就取消了；又说 OPC 社区有刻章打折，到了才知道是合作银行免费刻章。都是挺神奇的体验——方向大致对，细节得自己去核。

工作流这边也调了一下。把 AI Agent 接上了 Linear MCP，任务出口统一到 Linear；并把 Linear 配成了 Cycle（一周一个迭代）模式，用「当前周期」来表达"这周做什么"，比之前散在各处的待办清楚不少。

## blog 与 Homelab 杂项

blog 顺手做了两处改进：一是引入 Page Bundle 配图，文章和图片放在同一目录、随仓库进 git，给几篇 cloud 的文章加了封面图；二是接入 Pagefind 做站内搜索。

Homelab 这周是常规运维：把监控栈 kube-prometheus-stack 升到 87.2.1（带 Grafana 13.1.0）、ArgoCD 升到 3.4.4，顺手调了网关和 ingress。

## 咖啡

咖啡是另一条断断续续的线。最早在独立咖啡小店喝拿铁，喝出过肠胃问题（乳糖不耐，各家成分不一样）；后来改喝美式，又总觉得喝完心脏有点突突的，以为是个体差异。直到这周在星巴克点单，才注意到下单页面能选浓缩份数——美式默认是两份浓缩，拿铁只有一份。原来不是错觉，美式的咖啡因本就更高。这次有了经验，慢一点喝。

## 一个流油的大鸡腿

还是在星巴克。对面坐着个十岁左右的小男孩，斜对面是位不认识的年轻女士，我们三个互不相识。小男孩把作业本从包里掏出来摆好，又掏出一个用铝箔托盘和保鲜膜包着的大鸡腿，小心地撕开包装、怕油沾到手，然后一边写作业一边啃。

我看着那个大鸡腿，心想要是让我吃，这会儿是真吃不下，太油腻了，大概是刚吃过早饭还不饿。斜对面那位女士也在看小男孩撕包装，悄悄掏出手机给鸡腿和小男孩拍了张照。我和她相视一笑，又各自低头忙自己的电脑。我自言自语了一句"年轻人长身体"，两人都听见了，小男孩从作业本上抬起头朝我嘿嘿一笑，对面的女士跟我交换了个眼神也笑了。

一个流着油的大鸡腿，让三个素不相识的陌生人建立了三秒钟的奇妙连接。这种事，大概是 AI 做不到的。
