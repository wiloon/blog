# Jakarta<span class="Apple-style-span" style="font-size: 13px; font-weight: normal;"> </span>

<div id="bodyContent">
  <div lang="zh-cn" dir="ltr">
    <p>
      <strong>Jakarta项目</strong>是在Apache软件基金会营运的开放源代码开发项目之一。开发着面向对象编程语言Java的程序库，框架等。
    </p>
    
    <p>
      Jakarta是Apache组织下的一套Java解决方案的开源软件的名称，它包括了很多子项目。Tomcat、Ant、Struts等等现在是Apache下的开源项目，也曾是Jakarta的关联项目。
    </p>
    
    <p>
      Jakarta的名称是想把与Jakarta关系非常深的爪哇岛关联起来。编程语言Java的命名源自这个岛的名字（印尼语：<strong>Jawa</strong>、英语：<strong>Java</strong>），而城市雅加达（Jakarta）正是这个岛上的第一大城市，也是印度尼西亚的首府。
    </p>
    
    <h2>
      子项目
    </h2>
    
    <p>
      其中，Jakarta项目所包括的相关工具、库以及框架等罗列如下：
    </p>
    
    <ul>
      <li>
        BCEL &#8211; 处理Java字节码的类库
      </li>
      <li>
        BSF &#8211; 脚本程序框架
      </li>
      <li>
        Cactus &#8211; 服务器端Java类测试工具框架
      </li>
      <li>
        ECS &#8211; The Element Construction Set is a Java API for generating elements for various markup anguages.
      </li>
      <li>
        HttpComponents- 超文本传输协议.
      </li>
      <li>
        JCS &#8211; 分布式缓存系统.
      </li>
      <li>
        JMeter &#8211; 压力测试工具
      </li>
      <li>
        ORO &#8211; Java类库，提供与Perl5兼容的正则表达式功能
      </li>
      <li>
        Regexp &#8211; 纯Java正则表达式包
      </li>
      <li>
        Slide &#8211; a content repository primarily using WebDAV.
      </li>
      <li>
        Taglibs &#8211; 一个代码库，用于支持开发定制化的JSP tag lib
      </li>
    </ul>
    
    <p>
      以前隶属于Jakarta项目，但现在作为Apache软件基金的单独项目，有：
    </p>
    
    <ul>
      <li>
        Ant &#8211; 构建工具
      </li>
      <li>
        Commons &#8211; 一组使用类的合集，主要作为Java标准库的补充
      </li>
      <li>
        HiveMind &#8211; a services and configuration microkernel
      </li>
      <li>
        Maven &#8211; 一个项目构建和管理工具
      </li>
      <li>
        POI &#8211; 一个纯java版本的函数库，用于操作Microsoft的常见文档格式，如Excel、Word、PowerPoint、Visio、Publisher、Outlook文件.
      </li>
      <li>
        Struts &#8211; 一种Web应用程序开发框架
      </li>
      <li>
        Tapestry &#8211; A component object model based on JavaBeans properties and strong specifications
      </li>
      <li>
        Tomcat &#8211; 服务器，提供JSP/Servlet相关容器类
      </li>
      <li>
        Turbine &#8211; web应用快速开发框架
      </li>
      <li>
        Velocity &#8211; 模板引擎
      </li>
    </ul>
  </div>
