## windows win10 激活查询
### 查询windows 激活状态

```bash
slmgr.vbs -dlv 显示: 最为详尽的激活信息，包括: 激活ID、安装ID、激活截止日期
slmgr.vbs -dli 显示: 操作系统版本、部分产品密钥、许可证状态
slmgr.vbs -xpr 显示: 是否彻底激活
```
