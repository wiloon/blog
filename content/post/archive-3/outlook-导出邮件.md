## outlook 导出邮件, 备份, archive
File > Options > Advanced > Export > Export to a file > Next > Outlook Data File (.pst) > Next
展开树状列表，选择要备份的目录, 勾选 "invlude subfolders"
点击 Next
选择存放导出的pst文件的目标目录  (Save exported file as:) 
点击 Finish
不设置 密码 点击 确定 
等进度条走完
