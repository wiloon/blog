## 一句话区别

| 词 | 核心感觉 | 像在做什么 |
| ---- | -------- | ---------- |
| **exploration** | 开放地看看有什么可能 | 逛、试、学、发现新方向 |
| **investigation** | 带着问题查清楚原因或真相 | 调查、排查、取证、找责任 |

两个词都有「去了解、去弄清」的意思，但 **目的、语气、常见场景** 不同。

## Exploration

- **方向**：往往还没有固定答案，允许试错、绕路。
- **动机**：好奇、学习、找机会、体验。
- **语气**：中性偏积极，像冒险、研发、人生或职业上的一个阶段。
- **常见搭配**：
  - space **exploration**（太空探索）
  - career **exploration**（职业探索）
  - **explore** new ideas / markets（探索新想法、新市场）

例句：

> I'm in a period of **exploration**—trying side projects with AI.
>
> 我在一段探索期，用 AI 试各种项目。

> **Exploration** often means you don't know the outcome yet.
>
> 探索往往意味着结果尚未确定。

## Investigation

- **方向**：通常已经有一个 **具体问题** 要回答（谁、何时、为何、怎么发生的）。
- **动机**：找事实、找原因、找责任，满足合规或安全要求。
- **语气**：更严肃、更正式，常和法律、事故、纪律相关。
- **常见搭配**：
  - police **investigation**（警方调查）
  - internal **investigation**（内部调查）
  - **investigate** a bug / incident / fraud（调查故障、事件、欺诈）

例句：

> We're **investigating** why the API returned 500 errors last night.
>
> 我们在查昨晚 API 为什么返回 500。

> The company launched an internal **investigation** into the data leak.
>
> 公司对数据泄露启动了内部调查。

## 对比例句

把 exploration 换成 investigation，语气会明显变掉：

| 原句 | 感觉 |
| ---- | ---- |
| a period of **exploration** | 一段尝试新路的时期 ✅ |
| a period of **investigation** | 像在「被查」或「一直在查案」❌ |

技术场景里同样如此：

- **explore** a new framework → 试用、了解新框架（开放）
- **investigate** a production incident → 排查线上事故（有明确问题要查清）

## 和相近词

| 词 | 侧重点 |
| ---- | ------ |
| **research** | 更系统、有方法；论文、市场调研常用 |
| **inquiry / enquiry** | 询问、打听；美式里 *inquiry* 也常指官方调查 |
| **discovery** | 强调「已经发现」；探索感弱于 exploration |

## 怎么记

- **Exploration** ≈ 地图上未标明的区域，**去看看**。
- **Investigation** ≈ 已有线索的案子，**要查明白**。

## 使用场景备忘

给 TickTick 倒数日或人生阶段起名时，**Exploration** 比 **Investigation** 更合适：前者像「自主探索」，后者容易联想到「被调查」或「故障排查」。

若既要中文「自主探索」又要一个短英文词，可以：

- 标题：**自主探索**
- 英文备注：**Exploration**

完整表达 career 语境时，可写 **self-directed exploration**（自主探索、自己定方向的探索）。
