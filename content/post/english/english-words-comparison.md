## upgrade 和 update 的区别

- **upgrade** 一般是指比较重要的升级，或者说是主要的、独立版本的升级，其中软件改变很大。比如从 Windows XP 系统 upgrade 到 Windows 7 系统。

- **update** 一般是指比较次要的升级或更新，软件改变比较小或者只是修复一些 bug。比如从 Windows 7 系统 update 到 Windows 7 SP1。

**总结**：upgrade 一般指质的变化，而 update 一般指量的变化。
