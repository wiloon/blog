## idea 快捷键, keymap

- alt + shift + up, move line up

| Key    | Command      |
| ------ | ------------ |
| ctrl+g | line/column  |
| ctrl+t | Class... |
| ctrl+]  | move caret to code block end |
| ctrl+alt=b |Go to Implementation(s), 接口的实现类|
