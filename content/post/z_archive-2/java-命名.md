## java 命名

包名,使用网络地址倒置,小写,如com.myssh.action。(RFC规定url不准带下划线)
  
类名,大写开头,驼峰写法,如LoginAction。
  
方法名,小写开头,驼峰写法,如getUser()。
  
变量名,小写开头,驼峰写法,如userId。
  
常量名,大写开头,下划线分割,如MAX_LENGTH。
