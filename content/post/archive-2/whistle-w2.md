# whistle, w2
whistle HTTP, HTTPS, Websocket debugging proxy tool in linux, like Fiddler for windows

### install whistle
    yarn global add whistle
    npm install -g whistle

### w2 command
```bash
w2 help
w2 start
w2 stop

```

把代理指向127.0.0.1:8899

浏览器访问 http://127.0.0.1:8899/#network
