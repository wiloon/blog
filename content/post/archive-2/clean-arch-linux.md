## 'clean arch  linux'

```bash
  
sudo pacman -R $(pacman -Qtdq)

pacman -Scc
  
rm -rf ~/.cache
  
sudo pacman -S rmlint # remove duplicate file
  
sudo pacman -S ncdu
  
sudo pacman -S filelight
  
```
