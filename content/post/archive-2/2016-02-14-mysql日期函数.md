一、MySQL 获得当前日期时间 函数
  
1.1 获得当前日期+时间（date + time）函数：now()
  
mysql> select now();

+&#8212;&#8212;&#8212;&#8212;&#8212;&#8212;&#8212;+
  
| now() |
  
+&#8212;&#8212;&#8212;&#8212;&#8212;&#8212;&#8212;+
  
| 2008-08-08 22:20:46 |
  
+&#8212;&#8212;&#8212;&#8212;&#8212;&#8212;&#8212;+

除了 now() 函数能获得当前的日期时间外，MySQL 中还有下面的函数：
  
current_timestamp()
  
,current_timestamp
  
,localtime()
  
,localtime
  
,localtimestamp &#8212; (v4.0.6)
  
,localtimestamp() &#8212; (v4.0.6)

这些日期时间函数，都等同于 now()。鉴于 now() 函数简短易记，建议总是使用 now() 来替代上面列出的函数。
  
1.2 获得当前日期+时间（date + time）函数：sysdate()
  
sysdate() 日期时间函数跟 now() 类似，不同之处在于：now() 在执行开始时值就得到了， sysdate() 在函数执行时动态得到值。看下面的例子就明白了：
  
mysql> select now(), sleep(3), now();

+&#8212;&#8212;&#8212;&#8212;&#8212;&#8212;&#8212;+&#8212;&#8212;&#8212;-+&#8212;&#8212;&#8212;&#8212;&#8212;&#8212;&#8212;+
  
| now() | sleep(3) | now() |
  
+&#8212;&#8212;&#8212;&#8212;&#8212;&#8212;&#8212;+&#8212;&#8212;&#8212;-+&#8212;&#8212;&#8212;&#8212;&#8212;&#8212;&#8212;+
  
| 2008-08-08 22:28:21 | 0 | 2008-08-08 22:28:21 |
  
+&#8212;&#8212;&#8212;&#8212;&#8212;&#8212;&#8212;+&#8212;&#8212;&#8212;-+&#8212;&#8212;&#8212;&#8212;&#8212;&#8212;&#8212;+

mysql> select sysdate(), sleep(3), sysdate();

+&#8212;&#8212;&#8212;&#8212;&#8212;&#8212;&#8212;+&#8212;&#8212;&#8212;-+&#8212;&#8212;&#8212;&#8212;&#8212;&#8212;&#8212;+
  
| sysdate() | sleep(3) | sysdate() |
  
+&#8212;&#8212;&#8212;&#8212;&#8212;&#8212;&#8212;+&#8212;&#8212;&#8212;-+&#8212;&#8212;&#8212;&#8212;&#8212;&#8212;&#8212;+
  
| 2008-08-08 22:28:41 | 0 | 2008-08-08 22:28:44 |
  
+&#8212;&#8212;&#8212;&#8212;&#8212;&#8212;&#8212;+&#8212;&#8212;&#8212;-+&#8212;&#8212;&#8212;&#8212;&#8212;&#8212;&#8212;+

可以看到，虽然中途 sleep 3 秒，但 now() 函数两次的时间值是相同的； sysdate() 函数两次得到的时间值相差 3 秒。MySQL Manual 中是这样描述 sysdate() 的：Return the time at which the function executes。
  
sysdate() 日期时间函数，一般情况下很少用到。

二、MySQL 日期时间 Extract（选取） 函数。
  
1. 选取日期时间的各个部分：日期、时间、年、季度、月、日、小时、分钟、秒、微秒
  
set @dt = &#8216;2008-09-10 07:15:30.123456&#8217;;

select date(@dt); &#8212; 2008-09-10
  
select time(@dt); &#8212; 07:15:30.123456
  
select year(@dt); &#8212; 2008
  
select quarter(@dt); &#8212; 3
  
select month(@dt); &#8212; 9
  
select week(@dt); &#8212; 36
  
select day(@dt); &#8212; 10
  
select hour(@dt); &#8212; 7
  
select minute(@dt); &#8212; 15
  
select second(@dt); &#8212; 30
  
select microsecond(@dt); &#8212; 123456

2. MySQL Extract() 函数，可以上面实现类似的功能：
  
set @dt = &#8216;2008-09-10 07:15:30.123456&#8217;;

select extract(year from @dt); &#8212; 2008
  
select extract(quarter from @dt); &#8212; 3
  
select extract(month from @dt); &#8212; 9
  
select extract(week from @dt); &#8212; 36
  
select extract(day from @dt); &#8212; 10
  
select extract(hour from @dt); &#8212; 7
  
select extract(minute from @dt); &#8212; 15
  
select extract(second from @dt); &#8212; 30
  
select extract(microsecond from @dt); &#8212; 123456

select extract(year_month from @dt); &#8212; 200809
  
select extract(day_hour from @dt); &#8212; 1007
  
select extract(day_minute from @dt); &#8212; 100715
  
select extract(day_second from @dt); &#8212; 10071530
  
select extract(day_microsecond from @dt); &#8212; 10071530123456
  
select extract(hour_minute from @dt); &#8212; 715
  
select extract(hour_second from @dt); &#8212; 71530
  
select extract(hour_microsecond from @dt); &#8212; 71530123456

select extract(minute_microsecond from @dt); &#8212; 1530123456
  
select extract(second_microsecond from @dt); &#8212; 30123456

MySQL Extract() 函数除了没有date(),time() 的功能外，其他功能一应具全。并且还具有选取‘day_microsecond’ 等功能。注意这里不是只选取 day 和 microsecond，而是从日期的 day 部分一直选取到 microsecond 部分。够强悍的吧！
  
MySQL Extract() 函数唯一不好的地方在于：你需要多敲几次键盘。
  
3. MySQL dayof&#8230; 函数：dayofweek(), dayofmonth(), dayofyear()
  
分别返回日期参数，在一周、一月、一年中的位置。
  
set @dt = &#8216;2008-08-08&#8217;;

select dayofweek(@dt); &#8212; 6
  
select dayofmonth(@dt); &#8212; 8
  
select dayofyear(@dt); &#8212; 221

日期 &#8216;2008-08-08&#8217; 是一周中的第 6 天（1 = Sunday, 2 = Monday, &#8230;, 7 = Saturday）；一月中的第 8 天；一年中的第 221 天。
  
4. MySQL week&#8230; 函数：week(), weekofyear(), dayofweek(), weekday(), yearweek()
  
set @dt = &#8216;2008-08-08&#8217;;

select week(@dt); &#8212; 31
  
select week(@dt,3); &#8212; 32
  
select weekofyear(@dt); &#8212; 32

select dayofweek(@dt); &#8212; 6
  
select weekday(@dt); &#8212; 4

select yearweek(@dt); &#8212; 200831

MySQL week() 函数，可以有两个参数，具体可看手册。 weekofyear() 和 week() 一样，都是计算“某天”是位于一年中的第几周。 weekofyear(@dt) 等价于 week(@dt,3)。
  
MySQL weekday() 函数和 dayofweek() 类似，都是返回“某天”在一周中的位置。不同点在于参考的标准， weekday：(0 = Monday, 1 = Tuesday, &#8230;, 6 = Sunday)； dayofweek：（1 = Sunday, 2 = Monday, &#8230;, 7 = Saturday）
  
MySQL yearweek() 函数，返回 year(2008) + week 位置(31)。
  
5. MySQL 返回星期和月份名称函数：dayname(), monthname()
  
set @dt = &#8216;2008-08-08&#8217;;

select dayname(@dt); &#8212; Friday
  
select monthname(@dt); &#8212; August

三、MySQL 日期时间计算函数
  
1. MySQL 为日期增加一个时间间隔：date_add()
  
set @dt = now();

select date_add(@dt, interval 1 day); &#8212; add 1 day
  
select date_add(@dt, interval 1 hour); &#8212; add 1 hour
  
select date_add(@dt, interval 1 minute); &#8212; &#8230;
  
select date_add(@dt, interval 1 second);
  
select date_add(@dt, interval 1 microsecond);
  
select date_add(@dt, interval 1 week);
  
select date_add(@dt, interval 1 month);
  
select date_add(@dt, interval 1 quarter);
  
select date_add(@dt, interval 1 year);

select date_add(@dt, interval -1 day); &#8212; sub 1 day

MySQL adddate(), addtime()函数，可以用 date\_add() 来替代。下面是 date\_add() 实现 addtime() 功能示例：
  
mysql> set @dt = &#8216;2008-08-09 12:12:33&#8242;;

mysql>
  
mysql> select date\_add(@dt, interval &#8217;01:15:30&#8242; hour\_second);

+&#8212;&#8212;&#8212;&#8212;&#8212;&#8212;&#8212;&#8212;&#8212;&#8212;&#8212;&#8212;&#8212;&#8212;&#8212;&#8212;+
  
| date\_add(@dt, interval &#8217;01:15:30&#8217; hour\_second) |
  
+&#8212;&#8212;&#8212;&#8212;&#8212;&#8212;&#8212;&#8212;&#8212;&#8212;&#8212;&#8212;&#8212;&#8212;&#8212;&#8212;+
  
| 2008-08-09 13:28:03 |
  
+&#8212;&#8212;&#8212;&#8212;&#8212;&#8212;&#8212;&#8212;&#8212;&#8212;&#8212;&#8212;&#8212;&#8212;&#8212;&#8212;+

mysql> select date\_add(@dt, interval &#8216;1 01:15:30&#8217; day\_second);

+&#8212;&#8212;&#8212;&#8212;&#8212;&#8212;&#8212;&#8212;&#8212;&#8212;&#8212;&#8212;&#8212;&#8212;&#8212;&#8212;-+
  
| date\_add(@dt, interval &#8216;1 01:15:30&#8217; day\_second) |
  
+&#8212;&#8212;&#8212;&#8212;&#8212;&#8212;&#8212;&#8212;&#8212;&#8212;&#8212;&#8212;&#8212;&#8212;&#8212;&#8212;-+
  
| 2008-08-10 13:28:03 |
  
+&#8212;&#8212;&#8212;&#8212;&#8212;&#8212;&#8212;&#8212;&#8212;&#8212;&#8212;&#8212;&#8212;&#8212;&#8212;&#8212;-+

date\_add() 函数，分别为 @dt 增加了“1小时 15分 30秒” 和 “1天 1小时 15分 30秒”。建议：总是使用 date\_add() 日期时间函数来替代 adddate(), addtime()。
  
2. MySQL 为日期减去一个时间间隔：date_sub()
  
mysql> select date\_sub(&#8216;1998-01-01 00:00:00&#8217;, interval &#8216;1 1:1:1&#8217; day\_second);

| date\_sub(&#8216;1998-01-01 00:00:00&#8217;, interval &#8216;1 1:1:1&#8217; day\_second) |

| 1997-12-30 22:58:59 |
  
+&#8212;&#8212;&#8212;&#8212;&#8212;&#8212;&#8212;&#8212;&#8212;&#8212;&#8212;&#8212;&#8212;&#8212;&#8212;&#8212;&#8212;&#8212;&#8212;&#8212;&#8212;-+

MySQL date\_sub() 日期时间函数 和 date\_add() 用法一致，不再赘述。另外，MySQL 中还有两个函数 subdate(), subtime()，建议，用 date_sub() 来替代。
  
3. MySQL 另类日期函数：period\_add(P,N), period\_diff(P1,P2)
  
函数参数“P” 的格式为“YYYYMM” 或者 “YYMM”，第二个参数“N” 表示增加或减去 N month（月）。
  
MySQL period_add(P,N)：日期加/减去N月。
  
mysql> select period\_add(200808,2), period\_add(20080808,-2)

+&#8212;&#8212;&#8212;&#8212;&#8212;&#8212;&#8212;-+&#8212;&#8212;&#8212;&#8212;&#8212;&#8212;&#8212;&#8212;-+

+&#8212;&#8212;&#8212;&#8212;&#8212;&#8212;&#8212;-+&#8212;&#8212;&#8212;&#8212;&#8212;&#8212;&#8212;&#8212;-+
  
| 200810 | 20080806 |
  
+&#8212;&#8212;&#8212;&#8212;&#8212;&#8212;&#8212;-+&#8212;&#8212;&#8212;&#8212;&#8212;&#8212;&#8212;&#8212;-+

MySQL period_diff(P1,P2)：日期 P1-P2，返回 N 个月。
  
mysql> select period_diff(200808, 200801);

+&#8212;&#8212;&#8212;&#8212;&#8212;&#8212;&#8212;&#8212;&#8212;&#8211;+
  
| period_diff(200808, 200801) |
  
+&#8212;&#8212;&#8212;&#8212;&#8212;&#8212;&#8212;&#8212;&#8212;&#8211;+
  
| 7 |
  
+&#8212;&#8212;&#8212;&#8212;&#8212;&#8212;&#8212;&#8212;&#8212;&#8211;+

在 MySQL 中，这两个日期函数，一般情况下很少用到。
  
4. MySQL 日期、时间相减函数：datediff(date1,date2), timediff(time1,time2)
  
MySQL datediff(date1,date2)：两个日期相减 date1 &#8211; date2，返回天数。
  
select datediff(&#8216;2008-08-08&#8217;, &#8216;2008-08-01&#8217;); &#8212; 7
  
select datediff(&#8216;2008-08-01&#8217;, &#8216;2008-08-08&#8217;); &#8212; -7

MySQL timediff(time1,time2)：两个日期相减 time1 &#8211; time2，返回 time 差值。
  
select timediff(&#8216;2008-08-08 08:08:08&#8217;, &#8216;2008-08-08 00:00:00&#8242;); &#8212; 08:08:08
  
select timediff(&#8217;08:08:08&#8242;, &#8217;00:00:00&#8242;); &#8212; 08:08:08

注意：timediff(time1,time2) 函数的两个参数类型必须相同。

四、MySQL 日期转换函数、时间转换函数
  
1. MySQL （时间、秒）转换函数：time\_to\_sec(time), sec\_to\_time(seconds)
  
select time\_to\_sec(&#8217;01:00:05&#8242;); &#8212; 3605
  
select sec\_to\_time(3605); &#8212; &#8217;01:00:05&#8217;

2. MySQL （日期、天数）转换函数：to\_days(date), from\_days(days)
  
select to_days(&#8216;0000-00-00&#8217;); &#8212; 0
  
select to_days(&#8216;2008-08-08&#8217;); &#8212; 733627

select from_days(0); &#8212; &#8216;0000-00-00&#8217;
  
select from_days(733627); &#8212; &#8216;2008-08-08&#8242;

3. MySQL Str to Date （字符串转换为日期）函数：str\_to\_date(str, format)
  
select str\_to\_date(&#8217;08/09/2008&#8217;, &#8216;%m/%d/%Y&#8217;); &#8212; 2008-08-09
  
select str\_to\_date(&#8217;08/09/08&#8242; , &#8216;%m/%d/%y&#8217;); &#8212; 2008-08-09
  
select str\_to\_date(&#8216;08.09.2008&#8217;, &#8216;%m.%d.%Y&#8217;); &#8212; 2008-08-09
  
select str\_to\_date(&#8217;08:09:30&#8242;, &#8216;%h:%i:%s&#8217;); &#8212; 08:09:30
  
select str\_to\_date(&#8216;08.09.2008 08:09:30&#8217;, &#8216;%m.%d.%Y %h:%i:%s&#8217;); &#8212; 2008-08-09 08:09:30

可以看到，str\_to\_date(str,format) 转换函数，可以把一些杂乱无章的字符串转换为日期格式。另外，它也可以转换为时间。“format” 可以参看 MySQL 手册。

5. MySQL 获得国家地区时间格式函数：get_format()
  
MySQL get_format() 语法：
  
get_format(date|time|datetime, &#8216;eur&#8217;|&#8217;usa&#8217;|&#8217;jis&#8217;|&#8217;iso&#8217;|&#8217;internal&#8217;

MySQL get_format() 用法的全部示例：
  
select get_format(date,&#8217;usa&#8217;) ; &#8212; &#8216;%m.%d.%Y&#8217;
  
select get_format(date,&#8217;jis&#8217;) ; &#8212; &#8216;%Y-%m-%d&#8217;
  
select get_format(date,&#8217;iso&#8217;) ; &#8212; &#8216;%Y-%m-%d&#8217;
  
select get_format(date,&#8217;eur&#8217;) ; &#8212; &#8216;%d.%m.%Y&#8217;
  
select get_format(date,&#8217;internal&#8217;) ; &#8212; &#8216;%Y%m%d&#8217;
