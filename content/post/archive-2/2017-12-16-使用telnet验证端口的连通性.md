打开命令行模式。具体方法,请参考打开命令或 Shell 提示符 (2076587) (Opening a command or shell prompt (1003892))。
  
在命令行窗口键入: 

telnet server port

其中 server 是服务器的主机名或IP地址,port 是您想要连接的端口号。

按回车。
  
注: 要离开 Telnet 应用程序,请键入 Ctrl + ],然后键入 quit。
