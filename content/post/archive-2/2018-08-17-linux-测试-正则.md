使用 grep -P命令即可

```bash
echo 'a.gif' | grep -P '\.(jp?g|gif|bmp|png)'

#输出
a.gif
如果只想输出匹配部分,则加上-o参数

$ echo 'a.gif' | grep -P -o '\.(jp?g|gif|bmp|png)'

#输出
.gif
```
